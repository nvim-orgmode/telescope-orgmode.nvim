* INSTRUCTIONS: How to Use This Test

Welcome! This test verifies the core functionality of telescope-orgmode.

** How This Works
1. Each headline below is a test case with instructions
2. Follow the instructions to test the functionality
3. Check the checkbox [ ] when you complete each test
4. Save and exit (:wq) when done
5. The script will validate your results automatically

** Keybinding Reference
- =\th= - Open headlines picker
- =\tr= - Open refile picker
- =\ti= - Open insert link picker
- =<C-Space>= - Toggle between headlines and orgfiles mode
- =<C-f>= - Toggle current file filter
- =<CR>= - Confirm selection

** Ready to Start?
Press =\th= (backslash-t-h) to open the headlines picker and begin!
